# System Requirements

Jenkins is a web-based application that can be installed on any number of systems from laptops to servers.

1. **Hardware Requirements**
   1. At the minimum, you'll need at least **256MB** of RAM
   2. **1 GB** disk space

2. **Software Requirements**
   1. Java 8, either a **Java Runtime Environment** or **Java Development Kit** will work fine

If running on **Docker**

1. **Docker Hardware Requirements**
   1. **1 GB** of RAM
   2. **10 GB** disk space

2. **Docker Software Requirements**
   1. You need to have the Docker software installed (Docker Community Addition)
   2. Java not required to installed on local system. The pre-configured Docker images will already have java installed
